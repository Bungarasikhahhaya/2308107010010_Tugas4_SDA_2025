                                                SORTING ALGORITHM                       

                                                
Deskripsi
Proyek ini membahas dan menganalisis performa enam algoritma sorting terhadap dua tipe data: kata dan angka.
Pengujian dilakukan terhadap berbagai ukuran data (dari 10.000 hingga 2 juta item) untuk mengukur:
1. Waktu eksekusi (dalam detik)
2. Penggunaan memori (dalam byte)

Algoritma yang Diuji
1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort
6. Shell Sort

Detail Algoritma
1. Bubble Sort
- Kompleksitas: O(n²)
- Karakteristik: Perbandingan dan pertukaran elemen berulang-ulang.

2. Selection Sort
- Kompleksitas: O(n²)
- Karakteristik: Memilih elemen terkecil dan menempatkannya di posisi awal.

3. Insertion Sort
- Kompleksitas: O(n²)
- Karakteristik: Memasukkan elemen ke posisi yang tepat satu per satu.

4. Merge Sort
- Kompleksitas: O(n log n)
- Karakteristik: Pendekatan divide and conquer; membagi lalu menggabung.

5. Quick Sort
- Kompleksitas: O(n log n) rata-rata, O(n²) terburuk.
- Karakteristik: Divide and conquer; menggunakan pivot.

6. Shell Sort
- Kompleksitas: Bervariasi (rata-rata antara O(n log n) dan O(n²))
- Karakteristik: Versi upgrade dari Insertion Sort dengan gap interval.
